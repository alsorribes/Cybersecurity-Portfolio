# Comparación entre sistemas operativos habituales

En esta sección descubriremos algunos de los SO más conocidos.

## Sistemas operativos habituales

- **Windows y macOS:** Son SO muy comunes y se utilizan en ordenadores personales y empresariales. Windows es un SO de código cerrado, lo que significa que el código fuente no se comparte libremente con el público. En cambio. macOS es parcialmente de código abierto como por ejemplo su kernel.

- **Linux:** Es un SO completamente de código abierto y cualquiera puede acceder a él. Al ser así permite a los desarrolladores de la comunidad de Linux colaborar. Es un SO muy importante en la indústria de la seguridad ya que existen distribuciones diseñadas específicamente para ella.

- **ChromeOS:** Es parcialmente de código abiero y deriva de ChromiumOS. Se usa con frecuencia en el campo de la educación.

- **Android e iOS:** Son SO móviles para smartphones y tablets. Android es de código abierto y iOS es parcialmente de código abierto.
<br>

## Sistemas operativos y vulnerabilidades

Los problemas de seguridad son inevitables eb todos los SO. Por eso una parte importante es mantener actualizado el sistema.

- **SO heredados:** Es un SO obsoleto pero que se sigue utilizando. Algunas organizaciones siguen utilizando estos SO porque el software del que dependen no es compatible con los SO más recientes. Estos SO suelen ser más vulnerables porque ya no reciben soporte ni actualizaciones lo que están más expuestos a nuevas amenazas.
