# Interfaces GUI y CLI

### Interfaz gráfica de usuario (GUI)

Es una interfaz de usuario que usa iconos en la pantalla para administrar diferentes tareas en el dispositivo. La gran mayoría de SO pueden usarse con una interfaz gráfica de usuario. Estas interfaces suelen incluir los siguientes elementos:

- Menú de inicio con grupos de programas.
- Barra de tareas para iniciar programas.
- Escritorio con iconos y accesos directos.

Todos estos componentes ayudan a comunicarse con el SO para ejecutar las tareas. 
<br>

### Interfaz de línea de comandos (CLI)

Es una interfaz de usuario basada en texto que usa comandos para interactuar con el dispositivo. Estos comandos se comunican con el SO y ejecutan tareas. Tienen una estructura muy diferente a la GUI ya que esta no contiene ni iconos ni gráficos en la pantalla, aún así, es mucho más flexible y potente que una GUI. Por ejemplo, en estas interfaces se permite la personalización lo que le permite completar varias tareas simultáneamente.
<br>

### Ventajas de una CLI en ciberseguridad

El uso de una CLI en ciberseguridad puede proporcionar ciertas ventajas:

- **Eficacia:** Muchos prefieren la CLI porqué se puede utilizar más rápidamente si se sabe gestionar. Además, tiene más potencia para cuando necesita realizar múltiples tareas de forma eficiente. 

- **Archivo de história:** La CLI de Linux registra un archivo de história de todos los comandos y acciones usados en esta interfaz.
