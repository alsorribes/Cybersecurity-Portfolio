# Soporte integrado de Linux

### man

Este comando muestra información sobre otros comandos y su funcionamiento. Para usarlo se debe hacer de la siguiente manera: 'man chown', así obtendremos información sobre el comando 'chown' incluidas sus distintas opciones. 
<br>

### apropos 

Este comando busca en las descripciones de las páginas del manual una cadena específica. Las páginas de 'man' pueden ser largas y difíciles de buscar si lo que buscamos es una clave específica. Para usarlo lo haríamos de la siguiente manera: 'apropos 'inserte cadena''. 
<br>

### whatis

Este comando muestra la descripción de un comando en una sola línea. Por ejemplo, al introducir 'whatis nano' obtendremos la descripción de 'nano'.
