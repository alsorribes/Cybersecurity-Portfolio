# Distribuciones de Linux

Linux tiene diferentes versiones disponibles que se llaman distribuciones. Cada una tiene distintas herramientas y aplicaciones, por lo que cada distribución tiene una finalidad distinta a las otras. 

Las diferentes distribuciones de Linux contienen diferentes programas preinstalados e interfaces de usuario. Esto se basa en las necesidades del usuario de Linux.

La ventaja de usar Linux es que es muy personalizable. Todas las distribuciones incluyen el kernel, las utilidades, un sistema de administración de paquetes y un instalador. A partir de aquí se pueden crear nuevas distribuciones.
