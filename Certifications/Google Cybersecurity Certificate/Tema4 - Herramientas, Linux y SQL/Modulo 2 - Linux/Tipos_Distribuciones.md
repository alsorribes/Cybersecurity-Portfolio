# Tipos de distribuciones de Linux

### KALI LINUX

Es una marca comercial de Offensive Security y es un derivado de Debian. Esta distribución se hizo específicamente teniendo en cuenta las pruebas de penetración y la forense digital.

Una prueba de penetración es un ataque simulado que ayuda a identificar vulnerabilidades en sistemas, redes, sitios web, aplicaciones y procesos. KALI LINUX dispone de algunas herramientas muy útiles para estas pruebas, que son:

- **Metaesploit:** Se usa para buscar y explotar vulnerabilidades en máquinas.
- **Burp Suite:** Ayuda a probar puntos débiles en aplicaciones web.
- **John the Ripper:** Se usa para adivinar contraseñas.

El análisis forense digital es el proceso de recopilación y análisis de datos para determinar qué ha ocurrrido tras un ataque. En KALI también hay herramientas para este tipo de análisis, que son las siguientes:

- **tcpdump:** Analizador de paquetes de línea de comandos. Se usa para capturar tráfico de red.
- **Wireshark:** Tiene una interfaz gráfica de usuario que puede utilizarse para analizar en directo y capturar tráfico de red.
- **Autopsy:** Se usa para analizar discos duros y smartphones.

### Ubuntu

Distribución de código abierto, derivada de Debian y muy usada en seguridad y indústrias parecidas. Tiene tanto CLI cómo GUI. Viene con muchas aplicaciones por defecto pero los usuarios pueden descargar más desde un administrador de paquetes, incluidas herramientas enfocadas a la ciberseguridad. Esta distribución también se usa mucho para la computación en la nube.
<br>

### Parrot

Distribución de código abierto, basada en Debian y usada habitualmente en seguridad. Es parecida a KALI LINUX porque viene con herramientas preinstaladas relacionadas con pruebas de penetración y análisis forense digital.
<br>

### Red Hat Enterprise Linux

Distribución de Linux basada en una suscripción y diseñada para el uso empresarial. No es gratuita
<br>

### AlmaLinux

Distribución de Linux impulsada por la comunidad que fue creada como un reemplazo estable de CentOS.CentOS era una distribución de código abierto muy relacionada con Red Hat ya que usaba su código fuente para proprocionar una plataforma similar. AlmaLinux apareció para ser un reemplazo directo.
