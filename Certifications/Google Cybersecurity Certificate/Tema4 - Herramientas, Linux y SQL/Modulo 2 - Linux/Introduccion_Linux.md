# Introducción a Linux

Es un SO de código abierto lo que significa que cualquiera puede tener acceso al SO y al código fuente. Muchos de los programas que vienen con él están licenciados bajo los términos de la Licencia Pública GNU, que le permiten utilizarlos, compartirlos y modificarlos libremente. Gracias a esta filosofía de Linux, toda una comunidad de desarrolladores ha adoptado este SO y colaboran en proyectos. 

Linux se usa en diferentes organizaciones y más aún, en programas de seguridad. Como es de código abierto, muchos de los desarrolladores han creado nuevas distribuciones para contribuir en la comunidad.

En seguridad podemos usar Linux para hacer las siguientes tareas:

- Examinar diferentes tipos de registros para identificar qué está pasando en el sistema.

- Verificar el acceso y autorización en un sistema de gestión de identidad y acceso. Es clave para garantizar un sistema seguro.

- Trabajar con distribuciones específicas diseñadas para una tarea concreta.
