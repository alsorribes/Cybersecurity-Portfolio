# Comunicación en las redes

La comunicación a través de la red se produce cuando los datos se transfieren de un punto a otro. Estos fragmentos de datos suelen denominarse paquetes de datos. 

- **Paquete de datos:** Unidad básica de información que viaja de un dispositivo a otro dentro de una red. 

Cuando los datos se envían de un dispositivo a otro a través de una red, se envían como un paquete que contiene información sobre a dónde va el paquete, de dónde viene y el contenido del mensaje. 

El contenido de un paquete es el siguiente: 

- Encabezado que incluye la dirección del protocolo de Internet.
- Dirección IP i dirección MAC (Media Acces Control) del dispositivo de destino.
- Incluye el número de protocolo que indica al dispositivo receptor qué hacer con la información del paquete. 
- Contenido del paquete con el mensaje que debe transmitirse al dispositivo receptor. 
- Pie de página que indica al dispositivo receptor que el paquete ha finalizado. 

El rendimiento de una red puede medirse por el ancho de banda. 

- **Ancho de banda:** Cantidad de datos que puede recibir un dispositivo cada segundo. 

- **Latencia:** Es el tiempo que tarda un paquete en viajar de un punto a otro.
