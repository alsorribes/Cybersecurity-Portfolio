# Direcciones IP y comunicación en red

Una dirección IP es una cadena única de caracteres que identifica la ubicación de un dispositivo en Internet. Cada dispositivo en Internet tiene una dirección IP única que puede ser pública o privada. Tenemos dos tipos de direcciones IP: IP versión 4 o IPv4 y las IP versión 6 o IPv6.

### IPv4

Estas direcciones se escriben como grupos de 4 y en cada grupo hay 1, 2 0 3 dígitos separados por un punto decimal. Ejemplo: 192.168.1.1. A medida que Internet fue creciendo se agotaron las direcciones IPv4 y aparecieron las direcciones IPv6.

### IPv6

Estas direcciones se componen de 32 caracteres formadas por ocho números hexadecimales separados por dos puntos y cada número consta de hasta cuatro dígitos hexadecimales. Ejemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334. Esta longitud permitirá que más dispositivos se conecten a Internet sin quedarse sin direcciones tan rápido como en IPv4.

### Dirección MAC

Es un identificador de letras y números único que se asigna a cada dispositivo físico de una red. Cuando un conmutador recibe un paquete de datos lee la dirección MAC del dispositivo de destino y la asigna a un puerto. A continuación, guarda esta información en una tabla de direcciones MAC. Esta tabla es como una libreta de direcciones que el switch usa para dirigir los paquetes de datos al dispositivo apropiado.
