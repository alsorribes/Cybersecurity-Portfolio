# Redes

Una red es un grupo de dispositivos conectados, como por ejemplo: ordenadores, móviles o aparatos inteligentes. Estos dispositivos pueden comunicarse entre ellos a través de cables de red o conexiones inalámbricas. Una red de una casa y oficina pueden comunicarse con redes de otras ubicaciones y con los dispostivos que hay en ella. Para poder establecer comunicaciones los dispositivos se tienen que encontrar en una red. 

Todos ellos utilizan identificadores, como las direcciones IP y MAC, para poder localizarse y garantizar que las comunicaciones se producen con el dispositivo correcto. Hay dos redes en las que se pueden comuncar los dispositivos, que son: 

- **LAN (Local Area Network):** Red de área local que abarca una pequeña área como un hogar.

- **WAN (Wide Area Network):** Red de área extensa que abarca una área geográfica grande como una ciudad o un país. Internet sería como una gran WAN.
