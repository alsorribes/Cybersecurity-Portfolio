# Servidores proxy

Los servidores proxy son un tipo de servidor que se encuentra entre Internet y el resto de la red y que cumple con la solicitud de un cliente reenviándola a otros servidores. Cuando llegue una solicitud de conexión a la red desde Internet, el servidor proxy determinará si la solicitud de conexión es segura. 

El servidor proxy es una dirección IP pública que es diferente del resto de la red privada lo que hace que oculte la dirección IP de la red privada de los atacantes en Internet y añade una capa de seguridad. También se puede usar para bloquear sitios web no seguros a los que los usuarios no pueden acceder en la red de una organización. 

Hay varios tipos de servidores proxy:

- **Servidor proxy directo:** Regula y restringe el acceso de una persona a Internet.

- **Servidor proxy inverso:** Regula y restringe el acceso a Internet a un servidor interno. Su objetivo es aceptar el tráfico de terceros, aprobarlo y reenviarlo a los servidores internos. Esto es muy útil para proteger a los servidores web internos que contienen datos confidenciales para que no expongan su dirección IP a terceros.

- **Servidor proxy de correo electrónico:** Filtra el correo no deseado verificando si la dirección del remitente fue falsificada. Esto reduce el riesgo de ataques por suplementación de identidad.
